⚙️ Configure o Firebase primeiro

As credenciais ainda não foram preenchidas no arquivo. Siga os passos:

  1. Acesse console.firebase.google.com
  2. Clique em Adicionar projeto → dê um nome → crie
  3. Na tela do projeto, clique em Web (</>) → registre o app
  4. Copie o bloco firebaseConfig que aparecer
  5. Cole os valores em FIREBASE_CONFIG no topo do index.html
  6. No menu lateral: Build → Realtime Database → Criar banco
  7. Na aba Regras, cole: {"rules":{".read":true,".write":true}}
  8. Salve e reabra o site
